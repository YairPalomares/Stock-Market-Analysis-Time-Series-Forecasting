# Stock-Market-Analysis-Time-Series-Forecasting
Comparative stock market analysis of IBM and Walmart using Python (yfinance). Performed time-series decomposition (Trend/Seasonality) and stationarity testing via Augmented Dickey-Fuller (ADF). Implemented a 30-day Moving Average forecasting model, achieving a high-precision MAPE of 3.0% for WMT and 5.18% for IBM.
